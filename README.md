# 🔴 Who Is Tracking You?

A real-time browser privacy scanner that shows exactly what every website can see about you — without any login.

## What it reveals

- 📍 **Your IP Address** — live fetched
- 🗺️ **Approximate Location** — city, region, country
- 🌐 **Your ISP** — internet provider name
- 💻 **OS & Browser** — exact version
- 🖥️ **Screen Resolution** — color depth included
- 🔋 **Battery Level** — percentage + charging status
- 💾 **Device Memory & CPU Cores**
- 🍪 **Cookie & Storage tracking** — enabled or not
- 🔌 **WebGL GPU Fingerprint** — your graphics card
- 🔑 **Unique Browser Fingerprint** — tracks you across incognito
- 👁️ **Active trackers** — Facebook, Google, Amazon, ISP, Ad Networks

## Features

- 🔴 Privacy Risk Score (0-100) calculated in real time
- Matrix rain background animation
- Terminal-style UI with scan progress bar
- Share your score on Twitter/X
- 6 actionable protection tips

## Important

This page collects **zero data**.
Everything runs 100% locally in your browser.
No backend. No server. No tracking. Ironic, right?

## Tech Stack

HTML • CSS • Vanilla JavaScript
No frameworks. No dependencies.
